# ge

Usage
===========
ge [options ...]pattern-spec [files ...]pattern-spec [commit message ... ]

Purpose
===========
To simplify git commands

Options
===========
*-a*<br>
Add all files and commit them<br>
```ge -a [commit message]```
```  ```
*-f*<br>
Add a single file and commit<br>
```ge -f [filename] [commit message]```
